# Como desenvolver o Website

## Requisitos

Instalar Node Packet Manager (20.17.0) e npm (10.8.0) [Se for em Windows o msi vem com ambos, em linux não tenho a certeza) -> https://nodejs.org/en/download/prebuilt-installer

## Criar pasta de desenvolvimento

Alguns ficheiros são complicados de manter trabalhando no git diretamente, portanto:

-> Escolher localização que se queira

-> Abrir localização escolhida no terminal 

-> Executar comando npx create-react-app website_salva

-> Executar comando npm install gh-pages --save-dev

## Desenvolver (Repetir sempre que houver atualizações) 

-> Fazer Clone do Github

-> Copiar apenas as pastas src e public do Git para a pasta local website_salva 

-> Terminal na pasta local e correr o comando npm start

-> Quando terminar copiar apenas as pastas src e public para a pasta do git 

-> Fazer commit
